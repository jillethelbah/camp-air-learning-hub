Framework

What Are You Actually Making?

What shape is your idea? Find your form, then pick your tool. This evergreen progression helps you decide whether you need a one-time prompt, a reusable instruction, a repeatable skill, an automated workflow, an autonomous agent, or a full app with tools.

— Ben Weigel
PROMPT

You need it once. You ask it yourself, right now.

Level 1
Example: “Reword this email to sound friendlier.”
Basically any tool
INSTRUCTION

You want AI to always behave a certain way, without repeating yourself.

Level 2
Example: “Always reply in plain English — no jargon.”
M365 Copilot custom instructions GitHub Copilot in VS Code Copilot CLI Scout
SKILL

Same task, over and over — but you still kick it off each time.

Level 3
Example: A saved “weekly status email” you run every Friday.
M365/Bizchat declarative agent Copilot Studio Scout
WORKFLOW

Same job, runs without you — something triggers and it goes.

Level 4
Example: Every Monday at 9am: unread email becomes a digest.
Power Automate Copilot Studio with actions Scout GitHub Actions
AGENT

The steps are not fixed. You give the goal; AI finds the path.

Level 5
Example: “Plan the offsite” — it books the room and drafts the agenda.
Copilot Studio Scout Copilot CLI GitHub Copilot agent mode
APP WITH TOOLS

Needs a real interface — state, buttons, and reusable experiences others can open too.

Level 6
Example: A drag-and-drop priority board you reuse every week.
GitHub Copilot in VS Code Copilot CLI Power Apps
